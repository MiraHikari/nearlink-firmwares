## 报告漏洞

如果您发现安全漏洞，请不要创建公开的 issue。相反，请发送电子邮件至:

- mirahikari@gcxstudio.cn
- lmj-mc@outlook.com

请在邮件中包含以下信息:

1. 漏洞的类型
2. 复现漏洞的完整步骤
3. 受影响的版本
4. 可能的解决方案

我们承诺:

- 24小时内确认收到您的报告
- 及时通知您我们的处理进展
- 在修复后通知您
